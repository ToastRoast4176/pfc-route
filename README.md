# PFC Route

A Progressive Web App built for Peru Food Corp to digitize the food distribution workflow.

## Features

- Product catalog
- Customer management (in progress)
- Order creation (planned)
- Loading checklist (planned)
- Invoice generation (planned)
- Payment tracking (planned)
- Customer-specific pricing (planned)

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router

## Motivation

This project was created to replace a handwritten workflow used by Peru Food Corp, reducing errors while improving efficiency during customer visits, warehouse loading, invoicing, and payment tracking.
